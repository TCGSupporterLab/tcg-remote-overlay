# 修正内容の確認 (Walkthrough)

SPマーカーの表示・非表示をより柔軟かつ直感的に管理できるように修正しました。

## 主な変更点

### 1. 'O' キーによる表示トグル
- キーボードの **'O'** を押すことで、画面上のSPマーカーを即座に消したり出したりできるようになりました。
- これは「追従モード」「独立モード」のどちらを使用していても共通で動作します。

### 2. 非表示状態の自動リセット
- **オーバーレイの再起動時**: オーバーレイ画面を一度閉じて開き直したり、ブラウザを更新したりすると、非表示フラグが自動的にクリアされ、マーカーが表示される状態から始まります。
- **設定変更時**: 操作パネル側の「SP」バッジをクリックしてモード（非表示/追従/独立）を切り替えた際にも、非表示フラグがOFFになります。これにより「モードを切り替えたのに表示されない」といった混乱を防ぎます。

### 3. 操作画面の改善
- **ヒントの追加**: 画面左下の「SP」バッジにマウスを合わせると、ショートカットキーの説明が表示されるようになりました。
- **独立モードの修正**: 画面中央付近にマーカーが重なって表示されるバグを修正し、各モードが独立して正しくレンダリングされるように調整しました。

## テスト項目
- [x] 'O' キーでマーカーが消えること、もう一度押すと現れること。
- [x] マーカーを消した状態でオーバーレイ画面をリロードし、再表示されること。
- [x] マーカーを消した状態で「追従」から「独立」に切り替え、その瞬間に表示されること。
- [x] 遊戯王モードなど、他のモードのショートカットに影響がないこと。
