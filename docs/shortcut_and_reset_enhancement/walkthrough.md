# 修正内容の確認：ショートカットキーとリセット機能

## 変更の概要
キーボードショートカットの利便性を高め、マウスレスでの操作性を向上させました。また、試合の合間に素早く状態をリセットできる機能を追加しました。

## 修正詳細

### 1. ショートカットキーの追加・変更
- **`Shift + V`**: ビデオソースを逆順に切り替えます（画面共有 -> カメラ -> なし）。
- **`Alt + 1`**: 遊戯王モード（Yu-Gi-Oh! プリセット）に切り替えます。
- **`Alt + 2`**: ホロライブモード（hololive プリセット）に切り替えます。
- **`R` キー（1.5秒長押し）**: 全ての状態をリセットします。
  - 誤操作防止のため、長押しが必要になっています。
  - 長押し中にキーを離すとリセットはキャンセルされます。

### 2. 一括リセット機能 (Full Reset)
以下の項目が初期状態に戻ります：
- **ライフポイント**: `共通設定` で指定した初期ライフ（デフォルト8000）にリセットされ、ログもクリアされます。
- **ダイス**: 1 の状態にリセットされます。
- **コイン**: 表（1）の状態にリセットされます。
- **SPマーカー**: 表（front）の状態にリセットされます。

### 3. 設定画面のUI調整
- **共通設定セクションの移動**: 設定の最上部に移動し、アクセスの利便性を高めました。
- **リセットボタンの設置**: 共通設定内に「全ての状態をリセット」ボタンを設置しました。
- **見出しサイズの調整**: 「共通設定」「汎用ウィジェット」「専用ウィジェット」の見出しサイズを 20px に調整しました。

## 動作確認
- [x] `Shift + V` でビデオソースが逆順にループすること。
- [x] `Alt + 1` / `Alt + 2` で設定画面を開かずにモードが切り替わること。
- [x] `R` キーを1.5秒以上押し続けると、ライフやダイスがリセットされること。また、1.5秒未満で離した場合はリセットされないこと。
- [x] 設定画面の「全ての状態をリセット」ボタンで正しくリセットが行われること。
- [x] ガイドタブに新しいショートカットの説明が表示されていること。
