# 修正内容の確認: カード画像テンプレート配布機能の実装

カード画像の読み込みに使用するフォルダ構成のテンプレートと説明書を、UIからダウンロードできるようにしました。

## 変更内容

### 1. テンプレートアセットの作成
`public/assets/templates/` ディレクトリに、カード管理の雛形となるファイル群を作成し、zip形式でまとめました。
- **はじめにお読みください.txt**: 初心者向けの丁寧なガイド
- **hololive/**: ホロライブOCG用サンプル（`_meta_filter.json`, `_meta_card.json`, 実体/仮想フォルダ）
- **yu-gi-oh/**: 遊戯王OCG用サンプル（`_meta_filter.json`, `_meta_card.json`, 実体/仮想フォルダ）

### 2. UI（GuideTab.tsx）の更新
- 「セットアップ手順」の「カード画像を使う」セクションに、テンプレートのダウンロードボタンを追加しました。
- ページ下部に、将来の「GUIによる直感的なデッキ構築機能」の構想を記したロードマップセクションを追加しました。

## 検証結果
- `GuideTab.tsx` へのコンポーネント追加が正しく完了していることを確認しました。
- テンプレート配布用のzipファイルが指定の場所に正しく作成されていることを確認しました。
