# 修正内容の確認

## ファイル構成の変更
- **アーカイブ**: `docs/` が整理され、現在の仕様書とアーカイブフォルダのみになりました。
- **ツール隔離**: `tools/` フォルダが作成され、個人用の自動化スクリプトとデータがアプリのソースコード (`src/`) から分離されました。

## README.md の更新
- アプリのコンセプトが「ローカルのカード資産を活用する軽量オーバーレイ」であることを明確にしました。
- 内蔵データに関する古い情報が削除され、ユーザーに誤解を与えない内容になりました。

## コードの軽量化
- 未使用のフック `useWidgetSettings.ts` を削除し、ビルドサイズとメンテナンス負荷を軽減しました。
- `useWidgetStore.ts` 内の無意味なストレージキーを削除しました。

## 安全性の向上
- `.gitignore` の設定を強化し、個人用のカードデータやスクリプト、ダウンロード済み画像が誤って GitHub に公開されるのを確実に防止しました。
- `overlay_system.md` の通信詳細に、表示モード（Simple/Library）の同期が含まれていることを正確に記述しました。
