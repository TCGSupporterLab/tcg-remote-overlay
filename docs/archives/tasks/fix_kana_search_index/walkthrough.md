# 修正内容の確認 (Walkthrough)

カードの読み仮名（kana）が検索に反映されない問題を修正しました。

## 修正内容
これまではカードの「名前」に対してのみ検索インデックスを作成していましたが、データ内の「読み仮名」フィールドも検索対象に含めるように変更しました。

### 具体的な改善例
- **英名カード**: 
  - 名前 `Kotori` に対し、読み `ことり` がデータにある場合、「ことり」や「とり」で検索してもヒットするようになりました。
- **漢字名カード**:
  - 名前 `白上フブキ` に対し、読み `しらかみふぶき` がある場合、「しらかみ」等の平仮名入力でも確実にヒットします。

## 動作確認
- [x] 「ことり」で検索してカード `Kotori` が表示されること。
- [x] 「とり」などの部分一致でもヒットすること。
- [x] 他のカードでも、読み仮名による検索が機能していること。
