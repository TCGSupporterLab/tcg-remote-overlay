# カードタイプアイコン表示の実装計画

カード詳細オーバーレイの視認性とデザイン性を向上させるため、以下のUI変更を行います。

## 変更内容

### 1. カード名の配色変更
- 現在: カードの色（赤・青など）に応じた色で表示。
- 変更後: すべて「白」に統一。

### 2. カラータイプアイコンとHPの配置変更
- **現状**: [カードタイプ] ------------------- [タイプアイコン ＋ HP]
- **変更後**: **[タイプアイコン ＋ カードタイプ] ------------------- [HP]**
  - 左側に「タイプアイコン」と「カードタイプ（Debut等）」を並べて配置。
  - 右側には「HP/❤」のみを配置します。
  - これにより、カードの属性情報（色とレベル）が左側にまとまります。

## 対象ファイル
- `src/components/HololiveTools.tsx`

## 懸念点
- タイプアイコンは縦長（330x410）のものと、多色の小さいもの（88x108）が混在しているため、高さを固定して `object-fit: contain` で管理する。

## 検証プラン
- `推しホロメン`（単色）で正しくアイコンが表示されるか確認。
- `ホロメン`（多色：白緑など）で連結アイコンが表示されるか確認。
- サポートカードなどHPがないカードで、アイコンが表示されないことを確認。
