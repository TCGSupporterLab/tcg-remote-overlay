# ショートカットキーの拡充と一括リセット機能の実装

## 概要
ユーザーの利便性向上のため、以下の機能を実装しました。
- ビデオソースの逆送りショートカット (`Shift + V`)
- プリセット（遊戯王/ホロライブ）のクイック切り替え (`Alt + 1`, `Alt + 2`)
- 全状態の一括リセット機能（`R`キー1.5秒長押し、または設定画面のボタン）
- 設定画面（ウィジェットタブ）の構成見直しとリセット項目の追加
- 設定画面の大見出しサイズの調整 (20px)

## 実装内容
1. **`useWidgetStore.ts`**:
   - `fullReset` アクションを追加。ダイス、コイン、SPマーカーの状態を初期化し、`BroadcastChannel` を通じてLP計算機にリセット信号を送信します。
2. **`LPCalculator.tsx`**:
   - `BroadcastChannel` で `reset` メッセージを受信した際、ライフポイントと履歴を初期状態にリセットする処理を追加しました。
3. **`App.tsx`**:
   - `toggleVideoSource` に逆送り引数を追加。
   - `handleKeyDown` に `Shift + V`、`Alt + 1/2`、および `R`キー長押し（3秒）の判定を追加。
   - `R`キー長押しのタイマー制御のため `handleKeyUp` を追加し、長押し中のキャンセルに対応しました。
4. **`WidgetsTab.tsx`**:
   - 「共通設定」セクションを最上部に移動。
   - 「状態リセット」ボタンを「優先表示順」の上に追加。
5. **`GuideTab.tsx`**:
   - 使い方ガイドに新しいショートカットキーの説明を追加。
