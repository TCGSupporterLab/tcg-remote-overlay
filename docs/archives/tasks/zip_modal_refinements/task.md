# タスク: ZIPプレビューモーダルの洗練と展開・接続ロジックの修正

ZIPファイルのプレビューモーダルのUIを調整し、展開時に自動でサブフォルダを作成して即座に接続（スキャン）されるように修正する。また、Windows環境で発生していたエラーや、接続後の無限ループ問題を解消する。

## 完了した項目
- [x] ZIPプレビューモーダルのUI微調整（余白、ボタン配置、ラベルの整理）
- [x] モーダルから「主なフォルダ」セクションの削除（UIの簡素化）
- [x] 展開時に「ZIP名（拡張子なし）」のサブフォルダを自動作成するロジックの実装
- [x] 展開完了後、作成したサブフォルダをルートとして自動接続し、設定メニューを閉じる
- [x] Windows環境でのパス区切り文字（\）および禁止文字による `NotFoundError` の修正
- [x] 接続時・スキャン時の無限ループ（Blob URL의 再割り当て競合）の修正
- [x] 多重実行防止ガード（スキャン、展開）の導入
