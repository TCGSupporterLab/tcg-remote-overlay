# タスク: SPマーカー表示切り替えショートカットの実装とリセットロジックの改善

## 概要
ホロライブOCG用オーバーレイにおいて、SPマーカーの表示をキーボード操作（Oキー）で強制的に切り替えられるようにし、その状態がセッションごとに適切にリセットされる仕組みを構築する。

## 完了条件
- [x] 'O' キーでSPマーカーの表示・非表示が切り替わる（追従・独立両モード対応）。
- [x] 非表示状態は `localStorage` に保存せず、オーバーレイ再起動時にリセットされる。
- [x] 操作画面で「追従/独立/非表示」のモードを切り替えた際にも非表示フラグがリセットされる。
- [x] 操作画面のSPバッジにショートカットの案内を追加。
