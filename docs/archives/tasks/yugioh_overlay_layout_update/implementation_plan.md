# 遊戯王OCGオーバーレイのレイアウト変更および回転ボタンの削除

遊戯王OCG側のオーバーレイの見た目を、従来の横並びから「Player 2（180度回転）」「ダイス/コイン」「Player 1」の縦並びに変更します。また、操作画面における各プレイヤーの回転ボタンを不要との指示に基づき削除します。

## 変更内容

### 1. `src/components/YugiohTools.tsx` の変更

- **オーバーレイ時のレイアウト変更**:
    - `overlayRowClass` を使用していた箇所を、オーバーレイ時は `flex-col` になるように変更します。
    - 並び順を `Player 2` -> `OverlayDisplay` -> `Player 1` に変更します。
    - `Player 2` はオーバーレイ時、常に180度回転した状態で表示されるようにします。
- **操作画面の回転ボタン削除**:
    - `Player 1` および `Player 2` のカード右上に配置されていた `RotateCcw` ボタン（回転トグル）を削除します。
- **不要なスタイルの整理**:
    - 指示に応じて、不要になった回転関連のロジックやスタイルの適用を整理します。

## 影響範囲

- 遊戯王OCGの操作画面
- 遊戯王OCGのオーバーレイ画面

## 検証項目

- [ ] オーバーレイ画面で、上から P2(180度回転)、ダイス/コイン、P1 の順に並んでいること
- [ ] 操作画面で、各プレイヤーカードの右上にあった回転ボタンが消えていること
- [ ] ライフ計算等の基本機能に影響がないこと
