# タスク: SPマーカーの位置調整とカード角丸の改善

## 目的
- HololiveオーバーレイにおけるSPマーカーの表示位置を最適化し、文字表示モードでの見切れを防止する。
- カード画像特有の境界線（PNGの残り）を角丸トリミングで綺麗に隠す。
- 操作画面とオーバーレイ画面でデザインの一貫性を保つ。

## 完了条件
- [x] マーカーのオフセットを画像/テキストモードで個別に設定可能にする。
- [x] マーカーが親要素の `overflow: hidden` で切断されない構造に変更。
- [x] 全体の角丸を一括制御する定数 `OVERLAY_CARD_RADIUS` を導入。
- [x] 操作画面の詳細表示画像にも同様の角丸を適用。
- [x] 不要な境界線（グレーの線）が表示されないことを確認。
