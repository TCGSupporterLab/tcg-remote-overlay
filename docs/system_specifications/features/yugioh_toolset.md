# 遊戯王ツール仕様

## 1. ライフポイント管理 (LP Tracking)
- **初期値**: 8000
- **範囲**: 0 ～ 99999
- **履歴**: 直近10件の変更を各プレイヤーごとにログとして保持。

## 2. 計算機機能 (Calculator)
- **ターゲット管理**: Player 1 または Player 2 を選択（フォーカス）して操作。
- **演算**: 
    - `+`: 指定値を加算
    - `-` (または Enter): 指定値を減算
    - `÷2`: 現在のライフを半分（端数切り上げ）にする
    - `CLR`: 入力中の値をリセット
- **Undo / Redo**: Ctrl+Z / Ctrl+Y による履歴の戻し・進めに対応（最大履歴件数に準拠）。

## 3. ショートカットキー操作
操作を高速化するため、以下のキーボードショートカットをサポートしています。

| キー | 操作内容 |
| :--- | :--- |
| `0 - 9` | 数値入力 |
| `+` | 加算実行 |
| `-` / `Enter` | 減算実行 |
| `/` | ライフ半分 (÷2) |
| `Delete` / `Backspace` | 入力クリア |
| `↑` / `←` | Target を Player 1 に設定 |
| `↓` / `→` | Target を Player 2 に設定 |
| `Ctrl + Z` | 元に戻す (Undo) |
| `Ctrl + Y` | やり直し (Redo) |

## 4. ビジュアルとエフェクト
- **フォーカスエフェクト**: ターゲット中のプレイヤー枠が発光し、影（Shadow）が表示されます。
- **180度回転**: 対面操作を想定し、オーバーレイ表示時に各プレイヤーの数値を個別に180度回転させることが可能です。
- **ネオンデザイン**: オーバーレイ窓ではデジタル風の発光フォント（DigitalDisplay）を使用して視認性を高めています。
