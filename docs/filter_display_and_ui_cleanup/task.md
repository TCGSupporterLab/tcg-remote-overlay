# タスク：フィルタ表示の修正とUI整理

## 課題
1. `cardlist/hololive/all` などの第2階層以降に移動した際、フィルタリング項目が表示されなくなる。
2. フィルタ項目の抽出範囲が広すぎて、他のTCGの属性が混ざる可能性がある。
3. ローディングアニメーション（Scanning/Syncing）が `App.tsx` と `CardSearchContainer.tsx` で重複して実装されており、表示が重なる。

## 解決策
1. `useCardSearch.ts` の階層制限を緩和し、親フォルダの `_meta_filter.json` を遡って参照するロジックを実装。
2. フィルタ候補の抽出を現在のパス配下のカードのみに制限。
3. ローディング表示を `App.tsx` の共通コンポーネントに集約し、重複を削除。
