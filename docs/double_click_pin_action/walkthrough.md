# 修正内容の確認 (Walkthrough)

カードの管理（ピン留め）をより素早く行えるよう、ダブルクリックによるアクションを追加しました。

## 主な変更点

### 1. リスト上での操作
- カード検索結果、およびピン留め一覧のカード画像をダブルクリックすることで、ピン留めの登録・解除ができるようになりました。
- アイコンの小さなボタンを狙わなくても、画像全体がクリック対象となります。

### 2. 詳細画面での操作
- 画面左側の大きく表示されているカード画像をダブルクリックすることでも、ピン留めを切り替えることができます。
- 確認しながらピン留めしたい場合に便利です。

## 動作確認
- [x] カードをダブルクリックしてピン留めされること。
- [x] ピン留めされたカードをダブルクリックして解除されること。
- [x] シングルクリックでの「カード選択」機能が損なわれていないこと。
