# 修正内容の確認 (Walkthrough)

「オーバーレイ表示」のスイッチ（バッジ）がOFFの状態でも、SPマーカーの表示設定が優先されるように修正しました。

## 修正内容

### 1. 「追従モード」の挙動改善
- これまでは「オーバーレイ表示」がONでないとSPマーカーも表示されませんでしたが、これを分離しました。
- カード情報を隠したまま、裏・表の状態を示すマーカーだけを表示し続けることが可能です。

### 2. 「独立モード」の挙動維持
- 独立モードについては元々オーバーレイ設定に依存していなかったため、そのままの挙動を維持しています。

## 動作確認
- [x] 「オーバーレイ表示」バッジをOFFにする。
- [x] SPマーカーを「追従」または「独立」に切り替える。
- [x] カード画像は表示されないが、SPマーカーだけが正しく画面に表示されることを確認。
