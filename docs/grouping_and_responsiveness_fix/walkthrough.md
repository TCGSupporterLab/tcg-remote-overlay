# 修正内容の確認 (Walkthrough)

## 修正のポイント

### 1. 操作開始時のラグ解消 (Nonce 伝送)
親コンポーネントでの操作開始イベントを全子要素に Nonce (数値) 経由で通知するようにしました。これにより、ウィジェットが保持している「直前の操作による同期無視期間（150ms）」が、親（グループ全体）の操作が始まった瞬間に強制解除されます。
- **確認方法**: グループ化したウィジェットを素早くドラッグし、全要素が遅延なく一斉に動き出すことを確認。

### 2. 境界枠（BoundingBox）の表示最適化
複数選択（矩形選択や複数クリック）時に、個々のグループ用境界枠を表示しないようにしました。全体を包む「一時的な選択枠」のみが表示されます。
- **確認方法**: 2つ以上のグループまたは独立ウィジェットを選択した際、枠が重ならずスマートに表示されることを確認。

### 3. コインウィジェットの同期不具合修正
コインウィジェットが誤って SP マーカーの座標を参照していた問題を修正しました。
- **確認方法**: コインウィジェットが単独、またはグループの一部として正しく位置を記憶・同期することを確認。

---
これにより、特に「グループ化した要素の移動」において、ネイティブアプリに近いスムーズなレスポンスが実現されました。
