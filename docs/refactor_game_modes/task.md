# タスク: ゲームモード（遊戯王・ホロライブ）の廃止とマイレイアウトへの移行

## 概要
アプリケーション内の「遊戯王モード」および「ホロライブモード」というハードコードされた概念を完全に削除し、ユーザーが「マイレイアウト」機能を使用して自由に設定を管理できる構成に移行します。

- [x] 設定メニュー（`SettingsMenu.tsx`, `WidgetsTab.tsx`）から「遊戯王」「ホロライブ」のセクションを削除
- [x] `WidgetSections.tsx` から固有のセクションコンポーネントを削除し、汎用的な `LPSetting`, `SPMarkerSetting` にリファクタリング
- [x] 関連するドキュメント（システム仕様書等）の用語修正とファイル名変更
- [x] デフォルトレイアウトJSONファイルの名前を中立的なものに変更
- [x] 既存のLintエラーや重複コードの解消
