# タスクリスト: カードウィジェットの洗練と独立化

## 概要
ホロライブモードに統合されていたカード表示機能を独立した「カードウィジェット」として抽出し、ゲームモード（遊戯王/ホロカ）に関わらず利用できるようにします。また、不要となったテキスト表示機能を廃止し、UIをシンプルにします。

## タスク
- [x] **`useCardSearch` フックの修正**
    - `overlayDisplayMode` (テキスト/画像切替) 関連のステートとメソッドを削除
- [x] **`CardWidget` コンポーネントの新規作成**
    - `HololiveTools` からカード表示ロジックを抽出し、独立したコンポーネントとして作成
    - ダイスやSPマーカーを含まない、純粋なカード画像表示のみのウィジェットに
- [x] **`HololiveTools` コンポーネントの修正**
    - オーバーレイ表示モードを簡略化し、ダイス/コイン表示のみに限定
    - 未使用のコンポーネントや定数、テキスト表示ロジックを削除してコードをクリーンアップ
- [x] **`App.tsx` の更新**
    - メインオーバーレイ画面で `CardWidget` を独立してレンダリングするよう修正
    - 「カードを表示」設定がONの場合に、ゲームモードと並行してカードウィジェットを表示
    - 各ウィジェットを個別の `OverlayWidget` でラップし、個別に位置・サイズ・回転を調整可能に
- [x] **キャッシュ管理と表示ロジックの強化** (完了)
    - [x] `useCardSearch` に `displayCardNo` ステートを追加し、同期・永続化に対応
    - [x] `CardWidget` の表示ロジックを `No` とピン留め状態に基づいたものに刷新
    - [x] 何も表示するものがない場合のガイド表示を実装
    - [x] Shift+数字キー入力を `displayCardNo` の更新に紐付け
- [x] **不要なコンポーネントの削除**
    - `DisplayModeBadge.tsx` および `OverlayBadge.tsx` を削除
- [x] **`CardSearchContainer.tsx` の修正**
    - `CardWidget` から定数をインポートするように更新
