# 修正内容の確認 (Walkthrough) - テンキー「.」キーへのショートカット追加

テンキーの「.」キーを使用したダイスおよびコインのショートカット機能を実装しました。

## 変更の概要
- **`src/App.tsx`**
    - `lastDotTapRef`: 最後に「.」キーが押された時間を記録するための `useRef`。
    - `dotTimerRef`: ダブルタップ判定用のタイマー（`setTimeout`）を保持するための `useRef`。
    - `handleKeyDown`: 
        - `key` が `.` または `Decimal`、あるいは `code` が `NumpadDecimal` の場合を検知するように追加。
        - **遊戯王モード**:
            - 初回押下時に150msのタイマーを開始。
            - タイマーが切れる前に2回目が押された場合、タイマーをキャンセルしてコインを振る（`handleFlipCoin`）。
            - 150ms経過しても2回目が押されない場合、ダイスを振る（`handleRollDice`）。
        - **ホロライブモード**:
            - 即座にダイスを振る（タイマー待機なし）。
        - **共通**:
            - `e.preventDefault()` によりブラウザ既定の動作（通常は何も起きないが）を抑制。
            - 他のショートカット（`d`, `c` 等）と同様に、入力欄フォーカス時は処理をスキップ。
    - `useEffect` の依存配列に `gameMode` を追加し、モードに応じた挙動が正しく行われるように修正。
    - クリーンアップ関数でタイマーを確実にクリアし、予期せぬ実行を防止。

## 動作確認のポイント
1. **入力欄フォーカス時**: カード検索窓で文字入力していても、ダイスやコインが勝手に反応しない。
2. **通常/ホロライブモード時**: 「.」を押すと即座にダイスが振られる。連打すると連打した分だけ振られる。
3. **遊戯王モード時**: 
    - 「.」を1回押して一瞬待つと、ダイスが振られる。
    - 「.」をトントンと2回素早く押すと、ダイスではなくコインが振られる。

今回の修正により、片手での操作（テンキー操作）だけでもゲーム進行に必要なツールへアクセスしやすくなりました。
