# テンキー「.」キーへのショートカット追加

テンキーの「.」キーを使用してダイスやコインを振れるように機能を拡張します。

## 変更内容
- **ダイス機能**: テンキーの「.」（または`Decimal`キー）を押下するとダイスを振る。
- **コイン機能**: テンキーの「.」を300ms以内に2回素早く押下するとコインを振る（遊戯王モードのみ）。
- **フォーカス制御**: カード名検索などの入力欄にフォーカスがある場合は、他のショートカット同様に無視する。

## 実装の詳細 (App.tsx)
- `useRef` を使用して「.」キーの最終押下時刻と、ダブルタップ判定用のタイマーを管理します。
- `gameMode` が `yugioh` の場合のみダブルタップ判定（待機）を行い、それ以外（hololiveモードなど）では即座にダイスを振るようにします。
